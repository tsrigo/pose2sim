# Docs

`docs/` stores user-facing documentation that should remain useful after the
current milestone ends.

Current guides:

- `avi_to_trc.md`: end-to-end guide for the new batched AVI to 3D TRC CLI.

`context/` is intentionally separate. It stores milestone-local implementation
notes, smoke-test evidence, and PR-review material rather than stable user
guidance.
