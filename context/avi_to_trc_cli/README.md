# avi_to_trc CLI Milestone Context

This directory stores milestone-local implementation notes and local run
evidence for the `avi_to_trc` CLI. It is intended for PR descriptions and
review, not end-user onboarding.

Files:

- `README.md`: summary and evidence index
- `smoke-test.md`: concrete local run evidence from this milestone
- `pr-body.md`: PR-ready change summary and validation notes

Stable user guidance lives in `docs/avi_to_trc.md`.
